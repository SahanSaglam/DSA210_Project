# DSA210 Project – Movie Success Analysis

##  Project Description
This project analyzes factors influencing movie success using both financial and audience data. The goal is to understand how variables such as budget, genre, and popularity affect movie performance and to build predictive models.

##  Datasets
- TMDB 5000 Movie Dataset (Kaggle)
  - Includes budget, revenue, genres, and popularity
- IMDb Dataset
  - Includes ratings and number of votes

## Project Approach
The project combines the two datasets to analyze both financial success and audience perception of movies.

##  Analysis Plan
- Data cleaning and preprocessing
- Feature engineering (profit, ROI, success categories)
- Exploratory Data Analysis (EDA)
- Visualization of relationships between variables
- Correlation analysis
- Regression modeling

## Goals
- Identify key factors affecting movie success
- Compare financial success with audience ratings
- Build predictive models for movie performance
